/*
Primitive Data Types
---------------------------------------------------------------------------------------------------------
Type        Size            Range / Precision                          Example
---------------------------------------------------------------------------------------------------------
byte        1 byte          -128 to 127                                byte b = 100;

short       2 bytes         -32,768 to 32,767                          short s = 30000;

int         4 bytes         -2^31 to 2^31-1                            int num = 100000;

long        8 bytes         -2^63 to 2^63-1                            long l = 9999999999L;

float       4 bytes         6-7 decimal digits precision               float f = 10.5f;

double      8 bytes         15 decimal digits precision                double d = 99.999;

char        2 bytes         0 to 65,535 (Unicode values)               char ch = 'A';

boolean     JVM dependent   true or false                              boolean flag = true;
---------------------------------------------------------------------------------------------------------



Reference (Non-Primitive) Data Types
---------------------------------------------------------------------------------------------------------
Type/Class              Purpose                                        Example
---------------------------------------------------------------------------------------------------------
String                  Stores text                                    String name = "Ajay";

Array                   Stores multiple values                         int arr[] = {10,20,30};

Class                   Blueprint for objects                          class Student {}

Object                  Instance of class                              Student s1 = new Student();

Interface               Used for abstraction                           interface Payment {}

Abstract Class          Partial abstraction                            abstract class Vehicle {}

Enum                    Stores fixed constants                         enum Day {MONDAY}

Wrapper Classes         Object form of primitive types                 Integer num = 100;

Collections             Dynamic data structures                        ArrayList<String> list;
---------------------------------------------------------------------------------------------------------



Wrapper Classes
---------------------------------------------------------------------------------------------------------
Primitive Type              Wrapper Class
---------------------------------------------------------------------------------------------------------
byte                        Byte
short                       Short
int                         Integer
long                        Long
float                       Float
double                      Double
char                        Character
boolean                     Boolean
---------------------------------------------------------------------------------------------------------



Collections Used
---------------------------------------------------------------------------------------------------------
Collection                  Purpose
---------------------------------------------------------------------------------------------------------
ArrayList                   Dynamic array

HashMap                     Stores key-value pairs

HashSet                     Stores unique values only
---------------------------------------------------------------------------------------------------------

Primitive vs Reference Data Types
---------------------------------------------------------------------------------------------------------
Primitive Data Types                    Reference Data Types

Fixed size                             Size depends on object

Faster                                 Slightly slower

Cannot call methods                    Can call methods
---------------------------------------------------------------------------------------------------------



Examples
